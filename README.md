# WanderWay Travels 🌍
A responsive travel agency landing page built using HTML and CSS. 
This project showcases a modern, card-based layout for a fictional travel company, 
including destination highlights, travel packages, customer reviews, and a styled footer.

## Features
- **Hero Section** — Eye-catching banner with call-to-action button
- **Popular Destinations** — 4 destination cards with images, descriptions, and pricing
- **Travel Packages** — Detailed package cards with duration, price, and features
- **Customer Reviews** — Testimonial cards with photos and reviews
- **Responsive Footer** — Contact info and social media links

## Built With
- **HTML5** — Semantic page structure
- **CSS3**
  - Flexbox
  - CSS Grid
  - Media Queries (responsive design)
  - Box Shadow & Border Radius

## Responsive Design
Fully responsive across:
- **Desktop** (>900px) — Multi-column grid layout
- **Tablet** (600–900px) — 2-column layout
- **Mobile** (<600px) — Single-column stacked layout
